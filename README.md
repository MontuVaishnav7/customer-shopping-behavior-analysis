# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

Customer Shopping Behavior Analysis is an end-to-end Data Analytics project that explores customer purchasing patterns, subscription behavior, product performance, and revenue trends using retail transaction data.

The project demonstrates a complete analytics workflow including Data Cleaning, Exploratory Data Analysis (EDA), SQL Business Analysis, and Power BI Dashboard Development. The goal is to help businesses make data-driven decisions that improve customer retention, optimize marketing strategies, and increase revenue.

---

## 🎯 Business Problem

Retail businesses collect large volumes of customer transaction data but often struggle to convert it into actionable insights.

This project aims to answer important business questions such as:

- Which product categories generate the highest revenue?
- How do customer demographics affect purchasing behavior?
- Does subscription status influence spending habits?
- Which customer segments provide the highest business value?
- How can businesses improve retention and profitability?

---

## 🎯 Project Objectives

- Analyze customer demographics and purchasing behavior
- Identify high-value customer segments
- Evaluate subscription program effectiveness
- Discover top-performing product categories
- Build an interactive Power BI dashboard
- Generate actionable business recommendations

## 🔍 Key Findings

### Revenue Insights

- Clothing generated the highest revenue among all categories.
- Accessories were the second-largest contributor to revenue.
- Outerwear generated the lowest overall revenue.

### Customer Insights

- Young Adult customers generated the highest revenue.
- Subscription status influences customer engagement.
- Repeat buyers contribute significantly to business growth.

### Sales Insights

- Product category strongly impacts purchasing behavior.
- Discounts increase purchase activity but require optimization.
- Customer ratings help identify product quality and preferences.
